# MicroserviciosFSBlinblineo
EA2 Fullstack I DUOC Ingeniería en Informática

## Descripción del Proyecto
Sistema de microservicios para la gestión de inventario y pedidos de una tienda de ropa urbana, Blinblineo.cl.

## Microservicios Implementados
1. **Inventario-Service:** Gestiona el catálogo de productos (poleras, polerones, etc.) y su stock.
2. **Pedidos-Service:** Gestiona la creación de órdenes de compra y se comunica con Inventario para validar su disponibilidad.

## Tecnologías
* Java / Spring Boot
* Spring Data JPA & Hibernate
* OpenFeign (Comunicación entre servicios)
* Validación JSR-380 (Bean Validation 2.0)
